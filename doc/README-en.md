## 📌 About

**PDVShell** is a lightweight and efficient cash register (POS) system, developed in **Shell Script + SQLite**.

Ideal for small grocery stores that need something simple, fast and reliable.

---

## 🚀 Features

- 📦 Product registration and management
- 🏪 Supplier control
- 💰 Sales record
- 📊 Reports
- ⚙️ Settings
- 🧾 Inventory control

---

## 🧠 Philosophy

- Simple and direct
- No heavy dependencies
- Root terminal
- It even works on an old machine

---

## 📦 Installation

```bash
bash <(curl -s -L https://raw.githubusercontent.com/slackjeff/pdvShell/main/install.sh)
```

or

```bash
git clone --depth=1 https://github.com/slackjeff/pdvShell
cd pdvShell
./install.sh
```

---

## ▶️ Usage

```bash
pdvshell
```

or

```bash
./pdvShell
```
