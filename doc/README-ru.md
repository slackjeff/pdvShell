## 📌 О

**PDVShell** — это легкая и эффективная кассовая система (POS), разработанная на основе **Shell Script + SQLite**.

Идеально подходит для небольших продуктовых магазинов, которым нужно что-то простое, быстрое и надежное.

---

## 🚀 Особенности

- 📦 Регистрация и управление продуктами
- 🏪 Контроль поставщиков
- 💰Рекорд продаж
- 📊 Отчеты
- ⚙️ Настройки
- 🧾Контроль запасов

---

## 🧠 Философия

- Просто и прямо
- Никаких тяжелых зависимостей
- Корневой терминал
- Это работает даже на старой машине

---

## 📦 Установка

```bash
bash <(curl -s -L https://raw.githubusercontent.com/slackjeff/pdvShell/main/install.sh)
```

или

```bash
git clone --depth=1 https://github.com/slackjeff/pdvShell
cd pdvShell
./install.sh
```

---

## ▶️ Использование

```bash
pdvshell
```

или

```bash
./pdvShell
```
