## 📌关于

**PDVShell**是一个轻量级、高效的收银机（POS）系统，采用**Shell Script + SQLite**开发。

非常适合需要简单、快速和可靠的小型杂货店。

---

## 🚀 特点

- 📦 产品注册和管理
- 🏪 供应商控制
- 💰 销售记录
- 📊 报告
- ⚙️ 设置
- 🧾 库存控制

---

## 🧠 哲学

- 简单直接
- 没有严重的依赖性
- 根终端
- 它甚至可以在旧机器上运行

---

## 📦 安装

```bash
bash <(curl -s -L https://raw.githubusercontent.com/slackjeff/pdvShell/main/install.sh)
```

或者

```bash
git clone --depth=1 https://github.com/slackjeff/pdvShell
cd pdvShell
./install.sh
```

---

## ▶️ 使用方法

```bash
pdvshell
```

或者

```bash
./pdvShell
```
